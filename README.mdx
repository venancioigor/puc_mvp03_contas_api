# Meu Cofre API

Aqui você tem acesso ao BackEnd da minha API chamada Meu Cofre.

Foi construído utilizando o framework **Flask**.

Para executar o código, após clonar o aplicativo na sua máquina, instale todas as dependências, abra esse arquivo em um editor de sua preferência, no terminal navegue até a 
pasta raíz e execute o comando **python app.py** 

Assim, o servidor dessa API vai ficar disponível para receber requisições e enviar respostas dessas requisições,
tanto via Postman, Swagger ou requisições HTTP de um cliente Browser.

Para acessar a documentação dessa API, basta ir na URL http://127.0.0.1:5000/
quando o servidor estiver em execução.
